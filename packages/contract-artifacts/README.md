# @creativechain/contract-artifacts

Single shared source for CreativeChain smart-contract ABIs and deployment addresses.

## Source of truth

- ABIs are generated by `contracts/scripts/export-abis.js`.
- Amoy deployment addresses are generated by `contracts/scripts/deploy-amoy.js`.

## Usage

- Frontend imports from this package directly.
- Backend reads JSON files from this directory to avoid ABI duplication.
